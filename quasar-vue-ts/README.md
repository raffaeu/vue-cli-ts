# Quasar App

> A Quasar project migrated to use Typescript 2.0 and Tests from Vue templates *VUE-CLI*

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# test your app
$ quasar test

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
